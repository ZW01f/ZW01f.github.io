Let's start by detecting the profile 

```
$ vol.py -f MemoryDump_Lab6.raw imageinfo
```


![1.png](/assets/images/memlabs/lab6/1.png)

To list the processes that run on  the  system when this dump was taken , let's use [**pslist**](https://github.com/volatilityfoundation/volatility/wiki/Command-Reference#pslist) plugin 

```
$ vol.py -f MemoryDump_Lab6.raw --profile=Win7SP1x64 pslist
```


![2.png](/assets/images/memlabs/lab6/2.png)


I used **iehistory** plugin to see  internet explorer history .

```
$ vol.py -f MemoryDump_Lab6.raw --profile=Win7SP1x64 iehistory
```
